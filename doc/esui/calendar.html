2. 说明

日期类型的字符串格式为yyyy-MM-dd。

对于选中的日期，为Date类型的Javascript对象；对于可选择日期区间的数据格式，以一个Object表示。该Object拥有name为begin和end的属性，分别为Date类型。

{
    begin: new Date(1983, 8, 3),
    end: new Date(2011, 10, 4)
}
